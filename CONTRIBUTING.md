# Contributing to Marketplace Learn Monitor

Thanks for your interest in improving the Marketplace Learn Monitor!

## How to contribute

### Share feedback or request features

1. Open an issue on the [GitHub repository](https://github.com/jampick/marketplace-learn-monitor/issues)
2. Use the **Bug Report** or **Feature Request** template
3. Describe what you'd like clearly — the more context, the better

### What happens next

- Your issue lands in the **Backlog** on the project board
- The maintainer reviews and moves approved items to **Approved**
- Approved work is implemented on a feature branch, tested via CI, and merged via pull request

### Security

- Only the maintainer approves work items
- All changes require CI to pass before merging
- Pull requests are required for all changes to `main`
